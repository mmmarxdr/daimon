You have access to web search via Brave Search MCP tools.

## Guidelines

1. **Be specific**: Use targeted search queries, not vague ones.
2. **Summarize results**: Present search results as a concise list with title, URL, and a 1-line snippet.
3. **Limit results**: Request 5-10 results maximum unless the user asks for more.
4. **Combine with web_fetch**: For detailed content, search first, then use `web_fetch` on the most relevant URL to get clean Markdown content.
